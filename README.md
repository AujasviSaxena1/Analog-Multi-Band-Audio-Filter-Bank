# Active Multi-Band Audio Filter Bank

This filter bank, created using LTspice is used for the separation of an audio signal into different frequency bands, such as bass, midrange, and high-frequency components. For the same, Sallen-Key active filters are used, as well as a Twin-T notch filter for frequency rejection.

# Features

* Multi-band audio signal separation using different active filters 

* Implementation of Sallen-Key low-pass, band-pass, and high-pass filters

* Integration of a Twin-T notch filter for noise rejection

* Frequency response analysis using AC sweep analysis in LTspice.
